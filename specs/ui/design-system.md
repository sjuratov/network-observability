## Responsive Breakpoints

| Breakpoint | Name | Width | Layout Behavior |
|------------|------|-------|-----------------|
| `--bp-mobile` | Mobile | `< 768px` | Single column. Sidebar hidden (hamburger menu). Metric cards stack vertically. Tables become scrollable cards. Charts full-width. |
| `--bp-tablet` | Tablet | `768px – 1024px` | Sidebar collapsed to icons-only (56px) with hamburger expand. Metric cards 2×2 grid. Tables remain tabular with horizontal scroll if needed. |
| `--bp-desktop` | Desktop | `> 1024px` | Full sidebar (240px). Metric cards in a single row (4-up). Tables full-width with all columns visible. Side-by-side layouts where applicable. |

## Transitions & Motion

| Property | Duration | Easing | Usage |
|----------|----------|--------|-------|
| Background color | `150ms` | `ease` | Buttons, nav items, table rows on hover |
| Border color | `150ms` | `ease` | Input focus |
| Opacity | `200ms` | `ease-in-out` | Toast enter/exit, modal overlay |
| Transform (slide) | `200ms` | `ease-out` | Sidebar expand/collapse, dropdown open |
| Height (expand) | `250ms` | `ease-in-out` | Expandable table rows, accordion sections |

> **Reduced motion:** Respect `prefers-reduced-motion: reduce` — disable all transitions and animations when set.
